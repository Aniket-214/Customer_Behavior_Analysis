Customer_Behavior_Analysis

Customer Behavior Analysis is a data-driven project built using Power BI, SQL, and Python to understand customer spending patterns, demographic behavior, and revenue performance. The goal is to provide insights that help businesses identify valuable customer segments and make informed decisions.

✦ Project Highlights
✔ Data cleaned and transformed using Python (Pandas)
✔ Data extracted and analyzed using SQL queries
✔ Interactive Power BI dashboard developed to visualize customer trends
✔ Metrics analyzed: customer count, average purchase amount, review ratings, revenue by category, revenue by age group, and subscription vs non-subscription trends

✦ Tools Used
➤ Power BI for dashboard development and visualization
➤ SQL for data extraction and analysis
➤ Python for data cleaning and exploratory analysis

✦ Key Outcomes
✔ Identification of high-value customer segments
✔ Clear understanding of revenue patterns across demographic groups
✔ Insights that support better marketing, customer targeting, and business strategy
